# Vision — claw-cli

AI agent in your terminal. One command to chat, learn, and recall.

- `claw "question"` — one-shot
- `claw -i` — interactive REPL with memory
- `cat file | claw "summarize"` — pipe mode
- `claw --config` — configure API key

First validation product for the agentic ecosystem.
Proves that agentic-claw works as a runtime.
